Rocket Portfolio
================

Crypto portfolio tracker app by Lasse Juusela
Contact email: lasse.juusela@gmail.com

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

```
Next.js,
JavaScript,
Tailwind css,
ShadCN,
Coingecko.com APIs
```
